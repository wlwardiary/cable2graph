WikiLeaks Cablegate

Reference Network Visualization

A interactive visualization of the references between diplomatic cables released by WikiLeaks.

How is it made?

The cables.csv contains a field for referencing related cables via MRN. These 232319 references make a large network (or graph) with many groups of cables, also known as clusters. The largest cluster (giant) contains 91896 cables with 130574 references between them. Community detection can split the giant cluster into smaller communities.

From the total of 251287 cables ~56000 additional cables are referenced but not included in the cables.csv. These missing cables are also shown in the graph.

What can i do?

  • Zoom in and out of the graph with the (+ / -) on the upper left.
  • Show or hide labels and graph properties.
  • Move around the graph.
  • Jump directly to any cable listed in the sidebar.
  • Move over the cable id (MRN) to see the subject of the cable.
  • Click on any available cable id to read the full cable text on wikileaks.org.

How do i read the graph?

The visualization contains some indicators of graph and cable properties.

  • Cables with a red circle are not included in the cables.csv.
  • Node color shows the origin of the cable.
  • Node size indicate node degree.
  • First ring shows betweenness of nodes.
  • Second ring shows authority of nodes.
  • Line color indicates edge betweenness.

Browse the visualizations!

Please use a recent Chrome or Safari. Firefox does display it correctly, but is missing the "jump" functionality.
