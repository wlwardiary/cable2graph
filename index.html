The cables.csv contains a field for referencing related cables via ID. Not all the referenced ID's exist in the cables.csv. Some seem to be of higher classification and are not included. Some are typos. The missing cables are marked red.

If you can identify or guess the content of missing cables, please tweet with #wlfind #missing and the cable ID.

Use chromium / chrome / webkit. Use zoom.
