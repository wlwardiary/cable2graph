WikiLeaks Cablegate

Reference Network Visualization

The cables.csv contains a field for referencing related cables via MRN. These 232319 references make a large network (or graph) with many groups of cables (clusters). The largest cluster contains 91896 cables with 130574 references between them.

From the total of 251287 cables ~56000 additional cables are referenced but not included in the cables.csv

cables2graph provides a interactive visualization of this reference network with links to the original cables.
